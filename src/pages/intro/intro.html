The Ionic Component Documentation showcase a number of useful components that are include out of the box with Ionic.

What is Ionic?

Ionic Framework is an open source SDK that enables developers to build high quality mobile apps with web technologies like HTML, CSS and JavaScript.

What is Ionic Cloud?

The Ionic Cloud is a cloud plataform for managing and scaling Ionic apps with integrated services like push notifications, native builds, user auth, and live updating.
